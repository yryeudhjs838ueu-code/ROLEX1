# بوت حماية مجموعات تيليجرام من الإساءة 🛡️

هذا البوت مصمم لمراقبة المجموعات تلقائياً واستخدام الذكاء الاصطناعي لتحليل الرسائل. إذا قام أي عضو بإرسال شتائم، إهانات، أو إساءة للأديان، سيقوم البوت تلقائياً بـ:
1. **حذف الرسالة** المسيئة فوراً.
2. **كتم العضو** لمدة 3 ساعات (منعه من إرسال الرسائل).
3. **إرسال تنبيه** في المجموعة يوضح سبب الكتم.

## المتطلبات الأساسية
- الحصول على `Bot Token` من [@BotFather](https://t.me/BotFather).
- الحصول على `API Key` من OpenAI (أو استخدام البيئة المجهزة).
- رفع البوت كـ **مشرف (Admin)** في المجموعة مع صلاحيات:
  - حذف الرسائل (Delete messages).
  - حظر/كتم المستخدمين (Ban/Restrict users).

## كيفية التشغيل

### 1. تثبيت المكتبات اللازمة
```bash
pip install python-telegram-bot openai
```

### 2. إعداد متغيرات البيئة
يجب ضبط التوكن الخاص بك قبل التشغيل:
```bash
export TELEGRAM_BOT_TOKEN="your_telegram_bot_token_here"
export OPENAI_API_KEY="your_openai_api_key_here"
```

### 3. تشغيل البوت
```bash
python bot.py
```

## ملاحظات تقنية
- يستخدم البوت نموذج `gpt-5-mini` لتحليل النصوص بدقة عالية وفهم اللهجات العربية.
- يتم الكتم باستخدام خاصية `restrict_chat_member` لمدة 3 ساعات.
- البوت يتجاهل الرسائل من البوتات الأخرى والمشرفين (في حال لم يمتلك صلاحية كتمهم).
